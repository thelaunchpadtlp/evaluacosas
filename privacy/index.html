🛡️ Política de privacidad

Última actualización: 2026-05-06

Resumen for-dummies

  • Recolectamos solo los datos que necesitamos para que el docente pueda calificar tu entrega.
  • No usamos cookies de marketing, ni tracking de terceros, ni Google Analytics.
  • Tus datos se almacenan en infraestructura institucional de TLP.
  • Solo docentes autorizados pueden ver tus entregas.
  • Tus datos pueden ser borrados a pedido.

Qué datos recolectamos

Cuando hacés una evaluación:

  • Tu nombre completo (4 campos): para identificarte y registrar tu nota.
  • Email: cuando lo provees voluntariamente.
  • Cédula (opcional): solo si activás el lookup TSE.
  • Tus respuestas: el contenido que respondés en cada ítem.
  • Metadata técnica: timestamp, navegador, hash de tu sesión local.

Cuando un docente entra al dashboard:

  • Email institucional: @thelaunchpadtlp.education.
  • Nombre y foto de Google.
  • Token de sesión de corta duración (1 hora).

Dónde se guardan tus datos

Tu entrega se almacena con redundancia para no perderse:

  1. Google Cloud Storage (us-central1, EE. UU.) — proyecto institucional TLP.
  2. Firestore (us-central1) — base de datos NoSQL.
  3. GitHub privado (espejo cifrado) — repo institucional.
  4. Supabase (us-east-1) — base de datos relacional Postgres.
  5. Linear (workspace privado TLP) — tracking de revisión docente.

Por cuánto tiempo

  • Entregas activas: durante el ciclo lectivo en curso.
  • Archivo histórico: hasta 2 años después del cierre del curso, para auditoría académica.
  • Logs técnicos: hasta 30 días.
  • Cache de TSE: 24 horas máximo.

Quién puede ver tus datos

  • El docente del curso con cuenta institucional autorizada.
  • El administrador del Workspace TLP (joaquin.munoz@) por necesidad operacional.
  • Nadie más. No vendemos ni compartimos tus datos.

Servicios de terceros

  • Google Sign-In: para autenticar al docente. Sujeto a la política de privacidad de Google.
  • Google Cloud + Firestore: almacenamiento bajo el contrato Workspace TLP.
  • TSE de Costa Rica (a través de gometa.org o ApifyCR): solo si activás el lookup voluntario.
  • Gemini: solo cuando un docente pide calificación con IA. Se envía la entrega anónimamente al modelo. Google no entrena modelos con datos de Workspace.

Tus derechos

Podés pedir:

  • Acceder a una copia de tus datos.
  • Corregir datos incorrectos.
  • Borrar tu entrega (después de calificada y registrada).
  • Exportar tu entrega como JSON.

Escribí a joaquin.munoz@thelaunchpadtlp.education y respondemos en máximo 7 días.

Seguridad

  • HTTPS en todas las páginas.
  • HSTS Preload activado.
  • OAuth 2.0 con dominio restringido para docentes.
  • No se almacenan tokens en localStorage del navegador.
  • Auditoría de cada acceso al dashboard.

Cambios a esta política

Si hay cambios materiales, los publicamos en esta página y notificamos al Workspace TLP.
